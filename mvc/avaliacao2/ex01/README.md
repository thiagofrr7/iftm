Relatório do Exercício 1 - Thiago Ferreira Rocha

Primeiro foi colado na página "index.html" o código não estilizado. Em seguida, foram criados mais três páginas para que sejam referenciadas nos respectivos botões "Comprar" dos produtos. Para adicionar as imagens, peguei três imagens de camisa na internet que se adequassem a moldura já estabelecida no Bootstrap do Spring Boot e referenciei para a pasta "img". As três se encaixaram perfeitamente. Sobre as páginas de destino, copiei e reutilizei a div dos três produtos para criar uma página de comprar para os respectivos. A primeira div passou a mostrar a imagem do produto, a segunda a descrição e a terceira o preço e o cálculo do frete, seguido de um botão comprar.Não utilizei CSS, peguei as classes que já vieram com o exercício para aumentar ou diminuir a fonte dos títulos e subtítulos. Para o rodapé, utilizei a mesma dica dada em sala de aula: padding 3 e margin-top auto entre os atributos da classe do footer.
O display flex e flex column no body também ajudaram o footer a se fixar no fim da página.
